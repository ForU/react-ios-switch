React iOS Switch
===
[Demo](http://clariussystems.github.io/react-ios-switch)

```
<Switch
  className='switch' // optional
  checked={true}
  enabled={true} // optional
  onChange={...}/>
```

This is a [Clari](http://www.clari.com) open source project.

Setup
---
```
npm install
npm start
# Visit http://localhost:8080
```
