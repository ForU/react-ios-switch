react-ios-switch
===
Usage
---
```javascript
import Switch from 'react-ios-switch';

// Simple usage
<Switch
  checked={checked}
  onChange={({ checked }) => {}}
/>

// All props (default shown below)
<Switch
  checked={false}
  disabled={false}
  handleColor="#fff"
  offColor="#fff"
  onChange={() => {}}
  onColor="#4cd964"
  pendingOffColor="#dfdfdf"
  pendingOnColor={null}
/>
```
